**Copilot Researcher** is a pre-built agent focused on deep, multi-source information gathering and synthesis. 
*   **How it works:** It pulls and synthesizes information across web sources and your internal organizational documents.
*   **Best for:** This agent is ideal for strategy planning, knowledge workflows, and exploration. You should use it when you need to get a pulse on market trends, understand complex context, summarize research policies, or deliver detailed, comprehensive research reports.

**Copilot Analyst** is a pre-built agent designed specifically for processing and analyzing structured data.
*   **How it works:** It processes structured data formats like Excel or CSV files. A key differentiator is that the Analyst agent utilizes a built-in code interpreter (Python) to perform complex calculations and analyze the data.
*   **Best for:** This agent excels at pattern recognition, identifying numeric trends, and data processing. You should use it when you need to output numeric results, generate charts, create pivot tables, and provide natural language insights based on hard data.

**Custom Agents** are highly specific "mini-copilots" that you can build yourself to accomplish targeted business tasks or answer specific questions, such as an agent dedicated to answering employee queries about HR policies.
*   **How they are built:** You can create them without coding using the built-in Microsoft 365 agent builder, or use **Microsoft Copilot Studio** if you need deeper integrations, like connecting to external APIs.
*   **Configuration:** When creating a custom agent, you define its name, description, and instructions (which dictate its exact purpose, tone, and response structure). You can ground the agent by providing up to **20 knowledge sources** (such as specific Word documents, SharePoint folders, or websites) and configure up to **8 suggested prompts** to jumpstart user conversations.
*   **Add-on Capabilities:** You can also enhance custom agents by enabling specific add-on features, such as image generation or the ability to interpret Python code. 
The provided sources do not specify the exact monetary cost or pricing structure for using Copilot agents. However, they do detail the **licensing requirements** needed to access them:

*   **Additional License Required:** To create custom agents or utilize advanced pre-built agents like the Copilot Researcher and Copilot Analyst, you must purchase an "additional co-pilot license" on top of your standard Microsoft 365 subscription. 
*   **Basic vs. Advanced Access:** Unlicensed users (those with just a standard M365 subscription) only have access to basic chat features. Upgrading to the additional Copilot license gives you access to higher-tier language models, extra IT controls, and the specific ability to build and interact with agents.
