Based on the official study guide and the experiences of recent exam takers, the Microsoft AI Business Professional (AB-730) exam tests specific Microsoft 365 Copilot skills across three primary domains:

**1. Understand Generative AI Fundamentals (25–30%)**
*   **Copilot Capabilities:** Understanding how Copilot functions across different Microsoft 365 apps (Word, Excel, PowerPoint, Teams, Outlook) and knowing its specific capabilities and limitations within each. 
*   **Chat vs. Agents:** Differentiating between standard chat experiences and agent experiences, and knowing when it is appropriate to create a custom agent.
*   **Context and Grounding:** Understanding how your active work files, web data, or the specific app you are using affects Copilot's responses and the concept of Retrieval-Augmented Generation (RAG).
*   **Responsible AI:** Identifying risks such as AI fabrications (hallucinations), prompt injection attacks, and over-reliance on AI. You must also know how to apply verification steps like citation checks and human review.
*   **Data Security and Governance:** Understanding that Copilot strictly adheres to existing Microsoft 365 permissions and Microsoft Purview policies, meaning it will never access or surface data that a user does not already have permission to view.

**2. Manage Prompts and Conversations by using AI (35–40%)**
*   **Prompt Engineering:** Creating effective prompts by including a clear goal, context, resources/sources, and formatting expectations. This includes knowing how to save, schedule, and share these prompts with your team.
*   **Conversation Management:** Knowing the exact steps to find previous conversations, rename a chat, delete a chat, and add a conversation to a Copilot Notebook.
*   **Creating and Managing Agents:** Building custom Copilot agents using templates or Copilot Studio. You are tested on how to configure an agent's knowledge sources (up to 20 files or sites), define its instructions, set up suggested prompts, and share the agent with team members.

**3. Draft and Analyze Business Content by using AI (25–30%)**
*   **Document Drafting:** Using Copilot to create new documents from a prompt, generate documents based on existing files, create management summaries, and move data between different Microsoft 365 apps.
*   **Specialized Agents:** Knowing when to use the **Copilot Researcher** (for deep, multi-source information gathering and strategic reports) versus the **Copilot Analyst** (for structured data analysis, identifying trends, and generating charts from Excel/CSV files).
*   **Collaboration Tools:** Understanding the distinct use cases for **Copilot Pages** (lightweight, real-time collaboration built on Microsoft Loop) versus **Copilot Notebooks** (a workspace for complex, multi-step asynchronous projects and iterative prompting).
*   **Meeting Management:** Using Copilot in Microsoft Teams to summarize meetings and extract action items, which requires knowing that the meeting must be recorded/transcribed for Copilot to function. 
*   **Memory and Instructions:** Understanding how persistent custom instructions work, how Copilot utilizes temporary memory, and how it retains chat history (default 90 days).
